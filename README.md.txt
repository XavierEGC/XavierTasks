# XavierTasks - Organizador de Tareas

Una aplicación web para gestionar tareas diarias, con funcionalidades de priorización y filtrado.

![XavierTasks Screenshot](screenshot.png)

## Características

- ✅ Añadir, editar y eliminar tareas
- 🔢 Establecer prioridades (baja, media, alta)
- 🔍 Filtrar tareas (todas, activas, completadas)
- 💾 Almacenamiento local (las tareas persisten tras cerrar el navegador)
- 📱 Diseño responsivo para móviles y escritorio

## Tecnologías utilizadas

- HTML5
- CSS3 (Variables CSS, Flexbox)
- JavaScript (ES6+)
- LocalStorage API

## Cómo usar

Simplemente abre el archivo `index.html` en cualquier navegador moderno.

Para añadir una tarea:
1. Escribe la descripción de la tarea en el campo de texto
2. Selecciona la prioridad (por defecto es baja)
3. Haz clic en "Añadir Tarea" o presiona Enter
